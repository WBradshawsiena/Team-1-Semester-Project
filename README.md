## Controls:
### Player 1
    W - Jump
    A - Run left
    S - Shoot icicle
    D - Run right

### Player 2
    Up arrow - Jump
    Left arrow - Run left
    Right arrow - Run right
    Down arrow - Cast spear

### Miscellaneous
    Enter - Reset score
